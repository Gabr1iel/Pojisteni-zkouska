Vlastnosti aplikace

Aplikace je vytvořena v HTML 5 a stylovaná za pomocí CSS 3. Pro vytvoření funkcionalit stránky byl použit javascript a pro zjednodušení určitých částí Jquery. Stránka je responsivní jak pro desktop tak pro mobily. Více informací naleznete níže.

HTML 5

Kostra projektu napsána ve značkovacím jazyce HTML

CSS 3

Veškeré stylování vytvořené pomocí CSS

Některé animace dělané pomocí CSS

Responsivita

Stránka plně responsivní pro všechna zařízení

Použity dva breakpointy: 1024px a 480px

Upraven vzhled pro mobilní a menší desktopová zařízení

Javascript

Většina funkcí stránky implementovaná pomocí vanilla Javascriptu

Použita observer knihovna pro scroll animace

Jquery

Timeline implementovaná pomocí Jquery

Část data validation napsaná pomocí Jquery

Některé další části kodu zjednodušené pomocí Jquery
